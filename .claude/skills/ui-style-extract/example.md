# 示例输出（用户上传一张 Web3 金融 UI 截图后返回）

## 配色方案：
1. 总结：黑灰背景色，红色强调色，白色信息色。
2. 颜色HEX、透明度列表：
* 背景黑：#121217，100%，页面底层背景
* 面板灰：#1E1E29，100%，卡片容器
* 分割灰：#2D2D3B，100%，分割线
* 强调红：#D93A3A，100%，按钮、高亮
* 纯白：#FFFFFF，100%，标题、主要文字
* 弱灰文字：#AAAAAA，60%，辅助说明

## 字体：
1. 文字：Inter 字体，标题 700 加粗，正文 500 中等，辅助文字 400 常规，颜色白色或浅灰。
2. 数据：Inter 等宽数字，800 超粗，大号数字突出，右对齐，金融数据格式。

## 间距：
页面边距 24px，卡片内边距 16px~24px，元素间距 8px/12px/16px，卡片圆角 20px，按钮圆角 12px。
